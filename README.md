# Visualisierungen zu Integraltransformationen und partiellen Differentialgleichungen

Dieses Repository enthält Code, numerische Experimente und Visualisierungen zu Konzepten aus der Vorlesung **„Integraltransformationen und partielle Differentialgleichungen“**.

Ziel des Projekts ist es, mathematische Inhalte nicht nur formal, sondern auch **visuell und numerisch** besser verständlich zu machen.  
Das Repository ist in erster Linie als persönliches Lernprojekt gedacht, kann aber auch für andere Studierende hilfreich sein.

---

## Motivation

Viele Konzepte aus der Theorie partieller Differentialgleichungen und Integraltransformationen werden zunächst sehr analytisch eingeführt.  
Dieses Repository soll diese theoretische Sicht durch **Plots, Animationen und numerische Beispiele** ergänzen.

Die Idee ist nicht, die Theorie zu ersetzen, sondern zusätzliche **Intuition** für das Verhalten von Lösungen, Transformationen und Approximationen zu schaffen.

---

## Beispiele

### Fourierreihen

Partialsummen von Fourierreihen können benutzt werden, um Funktionen schrittweise zu approximieren.  
Besonders anschaulich wird das, wenn man betrachtet, wie sich die Approximation verändert, sobald mehr Terme hinzugenommen werden.
